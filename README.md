## Usage ‚å®Ô∏è

- Step 1 : Find group name from [here](https://docs.google.com/spreadsheets/d/1OF384yi-k3iBgiyLnhYDAoYAV8wJGCh2yEqm3MfQQko/edit?usp=sharing)

